Sahyadri Car's - Premium Luxury Inventory System

This is a professional SaaS-ready landing page for Sahyadri Car's, featuring a real-time car inventory management system and a customer delivery gallery powered by Firebase.

🚀 Step 1: Create Your GitHub Repository

Log in to GitHub.

Click the + icon in the top right and select New repository.

Name it sahyadri-cars.

Set it to Public (or Private if you prefer).

Click Create repository.

📂 Step 2: Organize Your Files

Before uploading, make sure your computer folder looks like this:

index.html (The code I generated for you)

src/ (A folder containing your brand logos)

mercedes.png

bmw.png

audi.png

porsche.png

jaguar.png

README.md (This guide)

📤 Step 3: Upload Your Code

In your new GitHub repository, click uploading an existing file.

Drag and drop the index.html file and the src folder into the window.

Scroll down and click Commit changes.

🌐 Step 4: Launch Your Website (Deployment)

To make your website live so customers can visit it:

Go to Vercel and sign in with your GitHub account.

Click Add New -> Project.

Import your sahyadri-cars repository.

Click Deploy. Vercel will give you a live link (e.g., sahyadri-cars.vercel.app).

🛠️ How to Manage Your Website

Visit your live website link.

Click the Login button in the navbar.

Since we are in "Preview Mode" for now, click Enable Manage Mode.

A floating button "MANAGE WEBSITE" will appear at the bottom left.

Use this dashboard to:

Add new cars to your inventory.

Delete old inventory.

Update the "Happy Deliveries" gallery with new customer photos.

⚠️ Important Note on Images

Logo Images: Ensure they are in the src folder and named exactly as shown above.

Car Uploads: When you upload a car photo via the dashboard, try to use images smaller than 1MB to keep the website loading fast for mobile users in India.
